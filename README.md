## Overview
This framework automates Virtual Service validation and control using a
configuration-driven, modular Python architecture.

## Features
- YAML-driven test configuration
- Parallel test execution
- Mock SSH and RDP connectors
- Pre-validation, task execution, and post-validation workflow
- REST API interaction using Bearer token authentication

## Setup
1. Install dependencies:
   pip install -r requirements.txt

2. Update credentials in config/credentials.yaml

3. Run:
   python main.py
